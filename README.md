Технологічний стек
Frontend: Angular 21, TypeScript, RxJS, Angular CLI
Backend: .NET 7 (або 8), Web API, Entity Framework Core
Database: MS SQL Server 

Ключовий функціонал

Обов'язкова автентифікація (Protected Identity):
Доступ до платформи є повністю закритим для неавторизованих гостей.
Реалізовано обов'язкову реєстрацію, безпечний вхід (Login) та вихід (Logout) з акаунту, а JWT-токени гарантують повну ізоляцію та приватність даних кожного користувача.

Повний життєвий цикл завдань (CRUD):
Безперешкодне створення, перегляд, редагування та видалення власних тасок.

Гнучке управління категоріями:
Можливість створювати та видаляти кастомні категорії для персоналізованого структурування робочого простору.

Розумний пошук та фільтрація:
Миттєва фільтрація списку завдань за обраними категоріями.
Ефективний пошук тасок за ключовими словами в описі завдання.

Архітектура та Clean Code:
Проєкт побудовано за принципами Clean Architecture (сервісний шар, розділена SPA-архітектура Angular + .NET Web API)
з фокусом на високу продуктивність та розширюваність коду.
